BackupForce

Desktop application for backing up and restoring Salesforce data using Bulk API v2. Supports CSV export and direct database integration.

Pre-built releases with bundled Java runtime. No additional software required.

Architecture

┌─────────────────────────────────────────────────────────────┐ │ BackupForce │ ├─────────────────────────────────────────────────────────────┤ │ UI Layer (JavaFX) │ │ ├── FXML Controllers │ │ └── CSS Theming │ ├─────────────────────────────────────────────────────────────┤ │ Service Layer │ │ ├── BackupService (5 parallel workers) │ │ ├── RestoreService (relationship resolution) │ │ └── TransformationService (cross-org mapping) │ ├─────────────────────────────────────────────────────────────┤ │ API Layer │ │ ├── BulkV2Client (Bulk API v2 + connection recovery) │ │ ├── SalesforceClient (REST API for metadata) │ │ └── OAuthManager (OAuth 2.0) │ ├─────────────────────────────────────────────────────────────┤ │ Storage Layer │ │ ├── CsvExporter │ │ ├── DatabaseExporter (Snowflake, PostgreSQL, SQL Server) │ │ └── BlobDownloader │ └─────────────────────────────────────────────────────────────┘

Key Components

Component Description
BulkV2Client HTTP client with connection pool recovery and version tracking
BackupService Multi-threaded orchestration with 5 parallel workers
RestoreService Insert/Update/Upsert with OldId → NewId mapping
TransformationService Cross-org RecordType, picklist, and user mapping

Requirements

Pre-built Releases

Windows 10/11, macOS 11+, or Linux (Ubuntu 20.04+, RHEL 8+)

Building from Source

Java 21+, Maven 3.9+

git clone https://github.com/victorfelisbino/BackupForce.git
cd BackupForce
mvn javafx:run

Configuration

Memory

Default: 4GB heap. Adjust for large objects:

java -Xmx8g -jar BackupForce.jar

OAuth

Configure Connected App in Salesforce Setup. See OAUTH_SETUP.md.
